# Checklist de Producción

## Pruebas PWA/offline

- [ ] La app carga con HTTPS.
- [ ] Manifest carga correctamente.
- [ ] Service worker se registra.
- [ ] Página /offline carga.
- [ ] Al apagar internet aparece banner sin conexión.
- [ ] Al intentar registrar asistencia offline muestra error claro.
- [ ] Al intentar crear reserva offline muestra error claro.
- [ ] Al volver internet aparece mensaje de conexión restablecida.
- [ ] No se cachean rutas administrativas.
- [ ] No se cachean datos privados.
- [ ] La app instalada abre correctamente.
